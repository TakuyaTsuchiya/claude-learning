参照情報：
- gh CLIとは：GitHub公式が提供するコマンドラインツール。ブラウザを開かずにターミナルからリポジトリの作成やプルリクエストの管理が可能。gitコマンド（ファイルの変更記録用）とghコマンド（GitHubサービス操作用）は別物で使い分けが必要。
- gh認証の手順：「gh auth login」→「GitHub.com」を選択→「Login with a web browser」を選択→表示されるワンタイムコードをブラウザに入力→「Authorize」で認証完了。
- リポジトリ作成からpushまでの流れ：mkdir→cd→git init→「gh repo create リポジトリ名 --public --source=. --remote=origin」でGitHub上にリポジトリ作成と接続設定が完了。その後git add→git commit -m "メッセージ"→git push -u origin mainの3ステップでアップロード。
- Claude Codeによる自動化：Claude Codeに「このプロジェクトをGitHubにプッシュして」と伝えれば、add・commit・pushを自動で実行してくれる。仕組みを知ったうえで自動化に任せるのが理想的な使い方。
