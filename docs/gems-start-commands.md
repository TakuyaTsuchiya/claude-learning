# 台本作成Gems 開始コマンド一覧

NotebookLMで該当セクションのソースを選択した状態で、台本作成Gemsに以下のコマンドをコピペして投げてください。1動画につき1コマンド。

## 使い方

1. NotebookLMで「Claude Code講座」ノートブックを開く
2. 左サイドバーで該当セクションのソース（ソースガイド + 準備物）を選択
3. 台本作成Gemsを開く
4. 以下の開始コマンドをコピペして送信
5. 出力された5テーブルを確認・保存
6. 次の動画の開始コマンドに進む
