## 本質：BPOの巻き取り

つまりこの講座が育てるのは「Claude Code × MCPで業務システムに直接つなぎ、BPOが人手でやっていた業務を全部自動化できる人材」。

BPOの典型業務：
- 業務システムにログイン→データ抽出→Excel加工→レポート作成→納品
- これが「MCPで接続→Claude Codeに指示→自動処理→PDF/Excel出力」に置き換わる

受講生がこの構造を理解すれば、どの業界のどの業務でも同じパターンで巻き取れる。

## 講座の核心メッセージ

**「リサーチ時間をゼロにする」**

どの業種でも仕事の大半は「調べる時間」に消えている。
- 営業 → 競合調査、顧客情報収集
- 法務 → 判例調査、契約条文の確認
- 経営者 → 市場調査、業界動向
- 投資家 → 銘柄分析、決算読み込み
- 経理 → 税制改正、勘定科目の判断
- 人事 → 候補者の経歴確認、制度調査

Claude Codeは「調べて→まとめて→判断材料にする」を一気通貫で回す。
人間は判断だけに集中できる。
