# Lessons — 構成作成時の学び

## 2026-04-14：「なぜ今やるのか」を過剰説明しない

### 指摘
S5-V3 Supabase のドラフトで「S5ではまだ使わないのに今セットアップする理由」を、重要トピック／受講者のつまずき／位置付け／シーンの流れの4箇所で繰り返し説明していた。土屋さんから「あとで使うので今セットアップしましょう、で当然のごとくやっていい」「『あとで慌てないために』はかえって意味不明、堂々としていて」との指摘。

### 学び
- **段取りとして自然なものは、過剰に理由づけしない**。「S6で使うので先に揃える」で十分
- **防衛的な説明は受講者の不安を逆に煽る**。「後で慌てないため」は「今やらないと慌てる」と読める
- **堂々とした語り口**：講師として「これが次の道具です、揃えましょう」と当然のごとく進める

### 適用方針
- 「なぜ今やるのか」を書きたくなったら、**それが受講者から実際に出る疑問か**を点検
- 「先出しセットアップ」「後で慌てない布石」「〜の前提」等の防衛的フレーズは削除候補
- シーンの流れで冒頭宣言する「なぜ重要か」は、**動画1本全体の意義**であり、個別段取りの理由づけではない
