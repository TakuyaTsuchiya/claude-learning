参照情報：
- Vercel CLIのインストール：npm install -g vercelでグローバルインストール。-gにより任意のフォルダからvercelコマンドが使える。permission errorの場合はMacならsudo npm install -g vercel、Windowsなら管理者として実行。vercel --versionでインストール確認。
- 認証フロー：vercel loginを実行し、GitHub/GitLab/Bitbucket/メールアドレスから認証方法を選択。ブラウザが自動で開き「Authorize Vercel」をクリックすれば完了。一度認証すればしばらく再認証不要。
- Claude Codeからのデプロイ手順：Claude Codeに「このプロジェクトをVercelにデプロイしてください」と指示するとvercelコマンドを実行してくれる。初回はSet up and deploy、プロジェクト名、スコープ選択、既存プロジェクトへのリンク、コードディレクトリ、フレームワーク自動検出などの質問に回答。2回目以降は初回の質問が省略される。
- デプロイ結果：プレビューURL（デプロイ固有）とプロダクションURL（本番）の2つが発行される。カスタムドメインはVercelダッシュボードからドメインを追加しDNS設定を行うことで設定可能。
